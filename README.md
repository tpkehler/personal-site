# Thomas Kehler Personal Site (GitHub Pages)

This folder contains a static personal website ready for GitHub Pages deployment.

## Quick publish steps

1. Create a new GitHub repository (for example: `tomkehler-site`).
2. Copy the contents of this folder into that repository root.
3. Commit and push to `main`.
4. In GitHub:
   - `Settings` -> `Pages`
   - Source: `Deploy from a branch`
   - Branch: `main` + `/ (root)`
5. Wait for the Pages URL to be generated.
6. Optional custom domain:
   - Set custom domain in Pages settings (for example `www.thomaskehler.com`)
   - Configure DNS records at your DNS host
   - Enable HTTPS when checks pass.

## Files

- `index.html` - complete single-page site (HTML + CSS in one file)

## Editing content

Open `index.html` and update:
- Hero statement
- About and timeline copy
- Writing section themes
- Contact links
